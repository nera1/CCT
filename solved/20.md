## Solution

- 입력값 n이 최대 200,000,000 으로 매우 크다
  - n 길이의 배열을 선언 또는 1부터 n까지 카운트 하는것만으로도 시간초과
    - 배열 선언을 못하면 최소값을 구하는 문제일지라도 DP문제 아니다

```java
class Solution {
    private int more = 0;
    private void recursive(int s, int e, int c) {
        if(s > e) {
            return;
        }
        if(e - s + 1 > c) {
            int mid = s + c - 1;
            recursive(s, mid, c);
            if(mid + 1 <= e) {
                recursive(mid + 1, e, c);
            }
        } else {
            more++;
        }
    }
    public int solution(int n, int[] stations, int w) {
        int start = 1;
        int end = 1;
        for(int s : stations) {
            int ss = s - w;
            int se = s + w;
            if(start >= ss && end <= se) {
                start = se + 1;
                end = se + 1;
                continue;
            }
            if(start < ss) {
                recursive(start, ss - 1, w*2 + 1);
                start = se + 1;
                end = se + 1;
                continue;
            }
        }
        if(start <= n) {
            recursive(start,n, w * 2 + 1);
        }
        return more;
    }
}
```
